# 最长公共子序列

$X = < A,B,C,B ,D , A , B >,Y = < B,D ,C,A ,B , A >$

那么序列 $< B,C,B,A >$ 就是 X 和 Y 的公共子序列

## 原理
$X_{m}$ 表示 $X$ 长度为 $m$ 的前缀
